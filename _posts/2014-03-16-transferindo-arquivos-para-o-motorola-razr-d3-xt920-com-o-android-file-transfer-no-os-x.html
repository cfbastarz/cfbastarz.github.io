Quando adquiri meu Motorola Razr D3, qual não foi minha decepção ao perceber que não conseguiria transferir minhas músicas e arquivos entre ele e o OS X utilizando o Android File Transfer? Depois de pesquisar por alguma solução, acabei escrevendo um post com algumas alternativas para sanar este problema, mas obviamente, nenhuma delas tão prática quanto ligar o telefone pela porta USB e transferir arquivos, assim como é possível fazer no Windows. No OS X, entretanto, esta tarefa não parecia ser tão simples assim... Mas eis que surge uma solução, e simples!

Para transferir arquivos para o Motorola Razr D3 (também conhecido como XT920) com o Android File Transfer no OS X, tive que instalar um app no Android chamado "DriveDroid". Este app serve para instalar baixar e instalar uma imagem do Linux no seu computador, servindo como uma alternativa de recuperação caso você não tenha opções em um momento de crise (muito útil!!) :D Mas o que interessa mesmo, é que este app possui uma interface para gerenciar e alterar a forma como o seu Android se comunica com o seu computador através da interface USB. Então, para resolver o problema de conexão que o Razr D3 possui como o OS X, vamos a um passa a passo. Mas antes de mais nada, é necessário que o seu telefone seja rooteado. Se você não sabe o que é isto, PARE POR AQUI. Eu não serei responsável por quaisquer danos causados ao seu telefone e qualquer tipo de perda de dados e informações. Faça por sua própria conta e risco.

A partir deste ponto, estou assumindo que o seu Motorola Razr D3 (XT920) é roteado , que você sabe o que está fazendo e que você é inteiramente responsável por isto.

1) Instale o DriveDroid a partir da PlayStore

[caption id="attachment_2308" align="aligncenter" width="288"] Instale o DriveDroid a partir da PlayStore (versão gratuíta)[/caption]

2) Se o root perguntar se você quer dar permissão para que o DriveDroid seja executado, permita

3) Abra o DriveDroid e clique no ícone de painel de controle (ajustes)

[caption id="attachment_2299" align="aligncenter" width="288"] Clique no ícone de ajustes (terceiro ícone na parte de baixo)[/caption]

4) Clique na opção "USB Settings"

[caption id="attachment_2300" align="aligncenter" width="288"] Selecione a opção "USB Settings"[/caption]

5) Clique em "Manually change permanent USB Mode"

[caption id="attachment_2301" align="aligncenter" width="288"] Selecione "Manually change permanent USB Mode"[/caption]

6) Selecione a opção "Media Transfer Protocol Android Debugging" e clique no botão "Use"

[caption id="attachment_2302" align="aligncenter" width="288"] Escolha a opção "Media Transfer Protocol Android Debugging"[/caption]

Após isto, você terá que habilitar a opção de deputação USB no seu telefone:

7) No App drawer do Android, clique no ícone "Configurar"

[caption id="attachment_2298" align="aligncenter" width="288"] Acesse as configurações do seu telefone[/caption]

8) Role a tela até a opção "Opções"

[caption id="attachment_2303" align="aligncenter" width="288"] Selecione "Opções"[/caption]

9) No alto, à direita, há um seletor "Sim/Não", certifique-se se que "Sim" está selecionado e assim obtenha acesso a configurações avançadas do Android

[caption id="attachment_2304" align="aligncenter" width="288"] Habilite a opção de "Depuração USB"[/caption]

10) Na mesma tela, role até a opção "Depuração USB" e selecione-a

A partir disto, seu telefone deverá ser reconhecido pelo OS X e ao plugá-lo no computador, o Android File Transfer (que deverá estar instalado no computador), deverá abrir automaticamente.

[caption id="attachment_2296" align="aligncenter" width="640"] Android File Transfer e Motorola Razr D3 no OS X (browser do armazenamento interno)[/caption]

[caption id="attachment_2297" align="aligncenter" width="640"] Android File Transfer e Motorola Razr D3 no OS X (browser do cartão SD)[/caption]

A princípio, esta mesma abordagem poderá ser utilizada em outros modelos/marcas de telefones com Android, desde que suportem o app DriveDroid.
